# 📊 Algoritma Sorting

Repository ini berisi implementasi beberapa algoritma sorting populer menggunakan bahasa Python, lengkap dengan pengujian performa pada berbagai kondisi data (*Best Case*, *Average Case*, dan *Worst Case*).

## 🚀 Tujuan Proyek

Proyek ini dibuat untuk mempelajari dan membandingkan efisiensi berbagai algoritma pengurutan data berdasarkan waktu eksekusi dan kompleksitas algoritma. Hasil pengujian dapat digunakan sebagai referensi dalam memahami kelebihan dan kekurangan masing-masing metode sorting.

## 📂 Algoritma yang Diimplementasikan

- 🔹 Bubble Sort
- 🔹 Selection Sort
- 🔹 Insertion Sort
- 🔹 Quick Sort

## ⚙️ Fitur

✅ Implementasi algoritma secara terpisah dan terstruktur  
✅ Pengujian pada data Best Case, Average Case, dan Worst Case  
✅ Pengukuran waktu eksekusi menggunakan modul `time`  
✅ Kode mudah dipahami dengan komentar penjelasan  
✅ Cocok untuk pembelajaran struktur data dan algoritma

## 📈 Kompleksitas Waktu

| Algoritma | Best Case | Average Case | Worst Case |
|-----------|-----------|-------------|------------|
| Bubble Sort | O(n) | O(n²) | O(n²) |
| Selection Sort | O(n²) | O(n²) | O(n²) |
| Insertion Sort | O(n) | O(n²) | O(n²) |
| Quick Sort | O(n log n) | O(n log n) | O(n²) |

## 🛠️ Cara Menjalankan

1. Clone repository:

```bash
git clone https://github.com/ValyaaDev/Algoritma-Sorting.git
```

2. Masuk ke folder project:

```bash
cd Algoritma-Sorting
```

3. Jalankan file Python yang diinginkan:

```bash
python bubble_sort.py
```

atau

```bash
python quick_sort.py
```
